# ViralLab 🌊

**AI Native 的 TikTok 内容增长操作系统，以 Claude Code Skills 形式实现。**

一条流水线，把品牌从"给我一个产品链接"带到"可以开拍的内容制作包"——全程 1 天。人只做两件事：定质量标准、拍策略决策；AI 执行其余一切：评论区全量抓取、人群洞察、爆款逆向拆解、品牌方级报告、脚本与分镜。

> English version: [README.md](README.md)

---

## 为什么做这个

传统 TikTok 增长调研的三个死结：

| 问题 | 常见现状 | ViralLab 的解法 |
|------|---------|----------------|
| 调研靠人肉 | 单品牌 2-3 周，只看几十条评论 | **1 天**，600+ 条 VOC，评论区抓到底 |
| 画像拍脑袋 | "25-35 岁精致女性"——零证据 | 每个画像必须附**"装备簇"**（这群人已经在用的真实商品）。形容词可以编，装备簇编不出来 |
| 经验不沉淀 | 教训留在运营脑子里 | 每次踩坑写回 SKILL 文件，**SOP 即代码，跨品牌复利** |

已在 2 个完整品牌案例上验证；第二个案例单日跑通全流程，并挖出多项人工调研不可能发现的洞察（见下方实战数据）。

## 系统架构

```
输入：一个产品名/链接
   ↓
Brand OS 品牌战略
  Step 0 产品档案（自动搜索）→ 社区定位（社区即画像草稿）
  → VOC 双线全量采集（本品+竞品评论区，滚到底）
  → 洞察金字塔（任务→意图→情绪→身份，全程用户原话支撑）
  → 输出：可外发的品牌策略报告 + 研究任务卡
   ↓
Research OS 爆款研究（7 步）
  关键词初扫（~150条）→ 时效/质量过滤 → 9 维评分选 Top 5
  → 真实逐字稿逐帧拆解 → 共同规律 → 品类 Viral DNA → 交接包
   ↓
Production OS 内容生产
  脚本（六段式公式）→ 分镜表 → 可直接粘贴的 AI 视频 Prompt
  → 剪辑指令 → 最小测试协议（A/B + 72h 数据决策）
   ↓
知识库（跨品牌复利）          交付层（在线报告页，单 HTML+图表，
品类 Viral DNA/Hook 公式库    可部署到任意静态托管）
```

## 快速开始

```bash
git clone https://github.com/sxhjas/viral-lab.git
```

作为 Claude Code plugin 加载，或把 `skills/` 拷进项目的 `.claude/skills/`。然后：

```
> 我想在 TikTok 上做 <产品链接> 的增长
```

Claude 会从 Brand OS Step 0 开始走完流水线。三个 skill 也可独立触发（丢一条 TikTok 链接直接进 Research OS 拆解）。

**推荐工作区结构**（skills 按此相对路径读写）：

```
products/{产品}/product_brief.md
brands/{品牌}/brand_brief.md + research_task_card.md + data_sources.md
projects/{项目}/research_output.md + production/ + session_state.json
knowledge/viral_dna.md + hook_formulas.md + comment_insights.md
```

## 五个差异点

1. **VOC 双线全量**：只抓本品评论=只听见买单的人；竞品评论区藏着"没选你的原因"。评论必须抓到不再加载为止——高赞常藏在懒加载后半段
2. **品-社区-品图谱**：人出现在哪个社区就说明她是谁；台面上的装备比问卷诚实。没有装备簇证据的画像直接打回
3. **真逐字稿拆解**：用官方 VTT 字幕/Whisper 转录，Hook 和 CTA 逐字引用，不看封面猜
4. **金字塔报告**：结论先行（一页摘要五个判断，每个带用户原话），方法论沉附录。品牌方 2 分钟读不懂就不合格
5. **SOP 自进化**：抓取工具失效、对比视频被评论区打假（249 赞实证）——修复方法和红线当场写回 skill

## 实战数据（案例二：某北美母婴清洁家电品牌，已脱敏）

- 单日 **615** 条有效 VOC（Reddit 全帖 + TikTok 本品/竞品评论区全量 + Amazon）
- 爆款漏斗 **150 → 19 → 5**，逐帧拆解
- 发现品类第一竞品是**用户家里已有的洗碗机**——而反驳弹药就在一条 2,112 赞的用户评论里
- 发现 **1390 万播放**的水安全大辩论——无品牌认领的流量池
- 目击达人摆拍对比被评论区 249 赞打假——沉淀为硬性红线：*对比必须公平可复现*

## 依赖说明
